# API de Estoque

Projeto desenvolvido com Spring Boot para gerenciamento de produtos.

## Tecnologias
- Java 17
- Spring Boot
- PostgreSQL

## Funcionalidades
- Criar produto
- Listar produtos
- Atualizar produto
- Deletar produto

## Como rodar
1. Configurar banco PostgreSQL
2. Rodar aplicação
3. Usar Postman

## Endpoints
POST /produtos  
GET /produtos  
PUT /produtos/{id}  
DELETE /produtos/{id}
